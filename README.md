# Agent-Based Simulation on Social Norms Change Interventions in Organizations

## Overview
This project simulates interventions to change social norms in organizations using an **agent-based model (ABM)** in R. 

- **`main.R`** – Runs the simulation by initializing agents, looping through the simulation, and storing results.
- **`config.R`** – Stores all simulation parameters. Modify the simulations parameters here.
- **`functions.R`** – Contains all the functions.
- **`results/`** – Stores output files such as simulation results (`.csv`) and generated plots.

To run the simulation, adjust parameters in `config.R` and execute `main.R`.
