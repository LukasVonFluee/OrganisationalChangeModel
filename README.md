# Agent-based simulation model on social norm change in organisations

This repository includes the R-scripts necessary to run the simulation. The scripts "config.R" and "functions.R" have to be executed before "simulation.R" can be run.

- **`config.R`** – Specifies the simulation parameters.
- **`functions.R`** – Specifies the functions used in simulation.R.
- **`simulation.R`** – Runs the simulation.
